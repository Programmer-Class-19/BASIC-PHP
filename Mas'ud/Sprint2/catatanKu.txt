Control Flow / Struktur Kendali 
    Control flow adalah konsep dalam 
    pemrograman yang mengatur urutan eksekusi dari instruksi atau pernyataan 
    dalam sebuah program. Dengan kata lain, control flow 
    menentukan bagaimana alur program berjalan dari satu
    bagian kode ke bagian lainnya, berdasarkan kondisi atau keputusan tertentu.
atau control flow itu  bagaimana alur dari programan kita dibaca inter prienter PHP.

Ada 2 bagain dalam mempelajari Control Kendali :
1. Pengulangan 
2. Pengkondisian

Bagian Pertama 

1. Control Flow Pengulangan
Gunanya menjalankan blok program / code tanpa perlu menulis ulang scriptnya . 
sintaks pengulangan :
1. for
2. while 
3. do.. while
4.foreach


Bagian Kedua 

2. Control Flow Pengkondisian 
Untuk menentukan apa yang akan terjadi ketika kita membuat sebuah pernyataan. 
Jika pernyataan hasilnya true kemudian apa yang akan terjadi dan juga sebaliknya 
jika pernyataan hasilnya false kemudian apa yang akan terjadi. 
sintaks pengkondisian :
1. if.. else
2. if.. else  if.. else
3.ternary
4. switch

Looping 
Looping, dalam pemrograman, adalah proses menjalankan 
sekelompok perintah berulang kali hingga kondisi tertentu 
terpenuhi atau selama kondisi tersebut masih benar.

looping biasanya dibatasi dengan tanda {kurung kurawal}

1. Built-in Function ( fungsi yang telah di sediakan php )               http://php.net/manual/en/funcref.php
2. User-Defined Function ( fungsi yang kita bikin sendiri )
